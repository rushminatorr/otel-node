# otel-node

Sample application from Opentelemetryjs using Exporter Collector.
[Exporter Collector](https://github.com/open-telemetry/opentelemetry-js/tree/master/packages/opentelemetry-exporter-collector) 

## Installation

```shell script
# from this directory
npm install
```

## Run the Application

```shell script
# from this directory
npm run
```
